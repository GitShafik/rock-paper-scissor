## Game: Rock Paper Scissor

---

## Game Dashboard

This is a simple Rock, Paper, Scissors game where you play against the computer. The game randomly selects one of the three moves—rock, paper, or scissors—while you choose your move by typing it in. The rules are straightforward:

- **Rock beats Scissors**  
  🪨✂️
- **Scissors beats Paper**  
  ✂️📄
- **Paper beats Rock**  
  📄🪨
- If both players choose the same move, it's a tie, and the game restarts.

### Instructions to play:

1. Run the script in a Python environment.
2. The game will prompt you to enter **rock, paper, or scissors**.
3. The computer randomly selects its move.
4. The result is displayed:
   - **You win!** 🎉 if your move beats the computer's.
   - **You lose!** 👎 if the computer wins.
   - **It's a tie!** 🔄 if both choose the same move.
5. If it's a tie, the game continues until there's a winner.

**OBS**: Enter a valid choice in order to play along, or you will be redirected to enter a valid choice.

**Best of luck! Enjoy the game!**
